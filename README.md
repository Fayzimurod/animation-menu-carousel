# 🍕 3D Pizza Slider - Interactive Product Showcase

A stunning, interactive 3D circular slider built with HTML, CSS, and JavaScript. This project creates an immersive product showcase with rotating images and synchronized content display - perfect for e-commerce product galleries or creative portfolios.


## ✨ Features

- **3D Circular Rotation** - Smooth, infinite rotation of product images with CSS transforms
- **Synchronized Content** - Product descriptions update automatically with image rotation
- **Responsive Design** - Works seamlessly across different screen sizes
- **Navigation Controls** - Previous/Next buttons with smooth transitions
- **Animated Content** - Text elements fade in with elegant animations
- **Visual Feedback** - Active states and hover effects for better UX
- **Auto-play Option** - Commented code available for automatic sliding

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS Variables and Keyframes)
- JavaScript (ES6+)
- Google Fonts (Pacifico)

## 📋 Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- 6 product images named 1.PNG through 6.PNG
- Basic understanding of HTML/CSS/JS (for customization)
